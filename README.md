# 🦕 Dino Runner

Десктопная игра в стиле динозаврика из Chrome, созданная на **Electron**.

## Структура проекта

```
dinosour_game/
├── main.js              # Electron main процесс (создание окна, жизненный цикл)
├── index.html           # HTML-структура игры
├── style.css            # Стили и анимации
├── renderer.js          # Логика игры (canvas, физика, частицы)
├── package.json         # Зависимости и скрипты
├── generate_icon.py     # Скрипт для генерации иконки
├── assets/
│   └── icon.png         # Иконка приложения (256x256)
└── README.md            # Этот файл
```

## Установка

### 1. Установка зависимостей

```bash
npm install
```

Это установит `electron` и `electron-builder`.

### 2. Запуск приложения (режим разработки)

```bash
npm start
```

Или с подробным логированием:

```bash
npm run dev
```

### 3. Сборка исполняемого файла (.exe, .dmg, .AppImage)

#### Для Windows (.exe):
```bash
npm run build:win
```
Результат: папка `dist/` с установщиком `.exe`

#### Для macOS (.dmg):
```bash
npm run build:mac
```
Результат: папка `dist/` с `.dmg` образом

#### Для Linux (.AppImage):
```bash
npm run build:linux
```
Результат: папка `dist/` с `.AppImage` файлом

> ⚠️ **Если AppImage не запускается:**
>
> **1. Установите библиотеку FUSE2** (самая частая причина):
> ```bash
> # Ubuntu / Debian / Linux Mint
> sudo apt install libfuse2
>
> # Fedora
> sudo dnf install fuse
>
> # Arch Linux
> sudo pacman -S fuse2
> ```
>
> **2. Сделайте файл исполняемым:**
> ```bash
> chmod +x dist/Dino\ Runner*.AppImage
> ```
>
> **3. Запустите:**
> ```bash
> ./dist/Dino\ Runner*.AppImage
> ```
>
> **Или через графический интерфейс:**
> - Кликните правой кнопкой по `.AppImage` → **Свойства**
> - Вкладка **Права** → ✅ **Исполняемый как приложение** (включите)
> - Закройте и дважды кликните по файлу
>
> **4. Если всё ещё не работает** — запустите с извлечением:
> ```bash
> ./dist/Dino\ Runner*.AppImage --appimage-extract-and-run
> ```

#### Сборка для всех платформ:
```bash
npm run build
```

> **Примечание:** Для кросс-компиляции могут потребоваться дополнительные зависимости.
> Рекомендуется собирать на той же ОС, для которой предназначена сборка.

## Управление

| Клавиша    | Действие                          |
|------------|-----------------------------------|
| `Пробел`   | Прыжок / Начать игру / Перезапуск |
| `Клик`     | Прыжок / Начать игру / Перезапуск |
| `R`        | Перезапуск после Game Over        |

## Функции игры

### Основные
- **Автоматический бег** — препятствия движутся справа налево
- **Гравитация** — реалистичная физика прыжка
- **Столкновения** — AABB проверка с уменьшенным хитбоксом (честнее)
- **Счёт** — увеличивается со временем
- **Рекорд** — сохраняется в `localStorage`
- **Увеличение скорости** — чем дольше бежите, тем быстрее
- **Стартовый экран** — "Press Space to Start"
- **Game Over экран** — с финальным счётом

### Дополнительные эффекты
- ✨ **Частицы** — при прыжке, беге и столкновении
- 📳 **Тряска экрана** — при проигрыше
- 🌗 **Смена дня/ночи** — по кнопке в левом верхнем углу

## Технологии

- **Electron 33** — фреймворк десктопных приложений
- **HTML5 Canvas** — отрисовка графики
- **requestAnimationFrame** — плавный игровой цикл (60 FPS)
- **CSS анимации** — тряска экрана, мигание текста
- **localStorage** — сохранение рекорда

## Настройки игры

В `renderer.js` можно изменить константы:

```javascript
const GROUND_Y = 320;          // Уровень земли
const GRAVITY = 0.6;           // Сила гравитации
const JUMP_FORCE = -12;        // Сила прыжка
const INITIAL_SPEED = 5;       // Начальная скорость
const SPEED_INCREMENT = 0.001; // Прирост скорости за кадр
const MAX_SPEED = 14;          // Максимальная скорость
```
